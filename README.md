# Atividades Monitoria

Este repositório tem como objetivo facilitar o compartilhamento de exercícios e materiais utilizados durante a monitoria, organizados por semana, com o objetivo de reforçar os conteúdos vistos em aula.

## Organização do repositório

Esta é a estruturação dos materias:  

```
semana-X/
   ├── exercicios.md
   └── respostas/
       ├── ex01.py
       ├── ex02.py
```

- Toda semana serão liberados de 5 a 6 exercícios
- As respostas serão liberadas no dia seguinte à monitoria
- Materiais adicionais solicitados pelos alunos serão adicionados em uma pasta separada
