# Semana 5

## Exercício 01

Faça um programa em que o usuário informe o salário recebido e o total gasto com despesas.
Deverá ser exibido na tela "Gastos dentro do orçamento" caso o valor gasto não ultrapasse o valor do
salário e "Orçamento estourado" se o valor gasto ultrapassar o valor do salário.

## Exercício 02

Faça um programa que ao receber o salário atual de um funcionário, calcule o valor de um novo salário reajustado,
de acordo com a tabela abaixo:

| Salário atual | Reajuste |
| ------------- | -------- |
| Abaixo de R$ 500,00 | 15% |
| de R$ 500,00 até R$1000,00 | 10% |
| Acima de R$1000,00 | 5% |

## Exercício 03

Dado um número inteiro n e as notas finais de n alunos, determine quantos alunos 
ficaram de recuperação. Um aluno está de recuperação se sua nota fina está entre 3.0 e 5.0, 
a nota máxima é 10.0.

## Exercício 04

Faça um programa em que o acesso do usuário só é permitido se a pessoa for administrador ou
funcionário da empresa. Utilize o operador or para verificar se o acesso é liberado.

## Exercício 05

Faça um programa que peça para o usuário digitar uma senha, ela será válida se possuir mais de 8 caracteres 
e não for igual a "123456". Utilize os operadores and e not.

## Exercício 06

Faça um programa onde uma loja oferece desconto caso o cliente faça uma compra acima de R$100,00 ou 
se o cliente for VIP. Utilize o operador or.
