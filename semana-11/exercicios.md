# Semana 8

## Exercício 01

Desenvolva um programa que crie uma lista vazia, adicione 5 números informados pelo usuário utilizando o método append (add), por fim imprima a lista final.

## Exercício 02

Crie um programa com uma lista inicial de nomes, peça ao usuário um novo nome e uma posição (índice).Utilize o método insert para adicionar o nome na posição informada, por fim exiba a lista atualizada.

## Exercício 03

Dada uma lista com vários números, peça ao usuário um valor específico, utilize o método remove para excluir a primeira ocorrência desse valor da lista. Mostre o resultado final.

## Exercício 04

Faça um programa que crie uma lista com 10 números inteiros, utilize um laço for para percorrer a lista e remover (com pop) todos os elementos que estão em índices pares, depois imprima a lista final.

## Exercício 05

Desenvolva um programa que possua uma lista com 5 elementos, peça ao usuário um índice, depois utilize o método pop para remover o elemento da posição informada. Mostre o elemento removido.

## Exercício 06

Crie um programa que permita ao usuário montar uma lista de tarefas.  
O usuário poderá:

- Adicionar uma tarefa (append)
- Inserir uma tarefa em uma posição específica (insert)
- Remover uma tarefa pelo nome (remove)

Ao final, exiba a lista completa.
